# TOPUP MAMA BACKEND CODE

### A backend Codebase written with Nestjs, Typescript, TypeORM, MySQL. 


## To get started

#### Clone Repository

#### cd into the repository

#### Install node modules
`npm install`

#### Setup Mysql DB

`start xampp`
`clone .env example`

#### Start server to test apis

`npm run start:dev`
